Other Projects

Java Projects

Animation

This project is about an animated day and night scene where the sun and moon move across the screen, accompanied by floating clouds, a house, trees, and flowers. The animation is achieved using a timer that updates object positions and repaints the panel at regular intervals.

Emotion Match

Emotion Match is a memory-matching game inspired by the animated film Inside Out. Players are presented with a grid of cards, each featuring imagery of emotions. The goal is to flip the cards, find matching pairs of emotions, and clear the board before time runs out.

Girl with a Pearl Earring

This project uses polygons and carefully selected color shades to recreate the famous painting "Girl with a Pearl Earring" by Johannes Vermeer. Each polygon represents a part of the painting, with precise coordinates and color blending to mimic the details of the artwork.
